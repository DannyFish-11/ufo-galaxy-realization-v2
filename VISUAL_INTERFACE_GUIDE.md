## 🚀 启动界面

### 方式一：启动完整系统

```bash
# 克隆仓库
git clone https://github.com/DannyFish-11/ufo-galaxy-realization-v2.git
cd ufo-galaxy-realization-v2

# 一键部署
./deploy.sh

# 启动系统
./start.sh
```

### 方式二：仅启动设备管理

```bash
# 启动设备管理服务
./start_device_manager.sh

# 或指定端口
./start_device_manager.sh 9000
```

## 📚 API 文档

### 访问地址

```
http://localhost:8080/docs
```

### 主要 API

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/devices` | GET | 获取设备列表 |
| `/api/devices/register` | POST | 注册设备 |
| `/api/devices/{id}` | DELETE | 注销设备 |
| `/api/execute` | POST | 执行命令 |
| `/api/status` | GET | 获取状态 |

## 🔧 自定义配置

### 修改端口

```bash
# 启动时指定端口
./start.sh --port 9000

# 或修改 .env
WEB_UI_PORT=9000
```

### 修改主题

编辑 `galaxy_gateway/static/device_manager.html`：

```css
/* 修改主题色 */
:root {
    --primary-color: #00d4ff;
    --secondary-color: #7b2cbf;
}
```

## 📱 移动端访问

### 局域网访问

```bash
# 获取本机 IP
ipconfig  # Windows
ifconfig  # Linux/macOS

# 手机访问
http://192.168.x.x:8080/devices
```

### Tailscale 访问

```bash
# 获取 Tailscale IP
tailscale ip

# 手机访问
http://100.x.x.x:8080/devices
```

## ✅ 使用流程

### 完整流程

```
1. 启动系统
   ./start.sh

2. 打开浏览器
   http://localhost:8080/devices

3. 注册设备
   - 选择设备类型
   - 输入设备名称
   - 输入 IP 地址
   - 点击注册

4. 管理设备
   - 查看设备状态
   - 发送控制命令
   - 注销不需要的设备

5. 监控系统
   - 查看控制面板
   - 检查 API 文档
```
